# İki Nokta Arası Uzaklık Hesaplayıcı
İstanbul Kent Üniversitesi Bilgisayar Programlama 2 ödevi için hazırlanmış C++ programı.


## 💻 Öğrenci Bilgileri : 
- **Ad Soyad :** Osman Yakut
- **Öğrenci Numarası:** 2507020079
- **Bölümü :** Bilgisayar Mühendisliği


## 👨‍💻 Programın çalışma Mantığı :
1) **cin** ile x1, x2, y1 ve y2 değişkenlerinin alınması
2) girilen değerlerden yola çıkarak **pow()** fonksiyonu ile "(x2 - x1)², (y2 - y1)²" kısımlarının hesaplanması
3) Hesaplanan iki sayının toplanıp d² değerinin elde edilmesi
4) **sqrt()** Fonksiyonu ile d²'nin karekökünün alınması ve d'nin yani iki nokta arası mesafenin elde edilmesi
5) Elde edilen sayının konsola yazdırılması 


## ⚙️ Program Özellikleri : 
- d² = (x2 - x1)² + (y2 - y1)² Formülünün C++ için programı.
- Sayı yerine bir karakter girilmesi durumunda programın bunu kabul etmeyip tekrar sayı istemesi.
- Görsel amaçlı duraksamalar.
